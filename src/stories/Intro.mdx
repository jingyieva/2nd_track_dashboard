# 二手交易分析儀表板 - Storybook

[🚀 專案 Demo](https://2nd-track-dashboard.vercel.app)  
[📖 Storybook Demo](https://2nd-track-dashboard.vercel.app/storybook/)

本 Storybook 展示了專案中的主要元件與場景，支援 Light / Dark 模式切換。

---

## 📦 已收錄元件

### UI 元件
- Button
- Badge
- Card
- Input
- Checkbox
- Select

### 功能元件
- Table（含排序 / Empty / Loading / Error 狀態）
- Pagination

### 圖表元件
- Bar / Line / Pie（含多序列、水平模式、Empty / Loading 狀態）

### 組合場景
- Table + Pagination

---

## 🚀 使用方式
- **本機**：`pnpm storybook` → [http://localhost:6006](http://localhost:6006)  
- **線上**：[Storybook Demo](https://2nd-track-dashboard.vercel.app/storybook/)

---

## 🔎 特點
- Controls 面板可即時調整 props（variant / size / 狀態）  
- 支援 Light / Dark Mode 切換  
- 提供 Empty / Loading / Error 狀態示例  
- 圖表可測試多序列堆疊、水平模式  
- 部分 stories 內建互動測試（play function）
